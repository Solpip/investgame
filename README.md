# investgame

Et enkelt investeringsspill i nettleseren.

## Om spillet

- Du starter med **1000 kr**.
- En **live graf** viser en aksjekurs som beveger seg tilfeldig opp og ned (oppdateres hvert sekund).
- **Kjøp** og **selg** aksjer for å prøve å øke formuen din.
- Tallene (kontanter, aksjer, aksjeverdi, totalformue og resultat) oppdateres i sanntid.
- Fremdriften lagres automatisk i nettleseren. Trykk **Nullstill spill** for å starte på nytt.

## Kjøre spillet

Åpne `index.html` i en nettleser — ingen installasjon nødvendig.
